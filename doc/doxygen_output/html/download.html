Download Boost.Geometry

The library can be downloaded from Boost SVN. There is anonymous SVN access.

The command

svn co https://svn.boost.org/svn/boost/sandbox/geometry geometry

will download the library to your machine.

The library can also be downloaded from Geodan SVN. This will be moved to osgeo SVN
