Detailed Description

disjoint means spatially disjoint, there is no overlap of interiors and boundaries, the intersection of interiors or boundaries is empty.

Geometries:
  • point + point (= ! equals)
  • point + box (= not within or on border)
  • box + box
  • ring + box
  • polygon + box
  • polygon + ring
  • polygon + polygon
