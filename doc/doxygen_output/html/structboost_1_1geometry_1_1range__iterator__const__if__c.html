Detailed Description

template<bool IsConst, typename Range>
struct boost::geometry::range_iterator_const_if_c< IsConst, Range >

Meta-function to define a const or non const boost range iterator.

Is used to have one implementation for both const and non const range iterators

Note:
This traits class is completely independant from GGL and might be a separate addition to Boost
