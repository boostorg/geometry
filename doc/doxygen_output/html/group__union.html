Detailed Description

The union of two geometries A and B is the geometry containing all points belong to either of A or B, but no other elements.
Source description:
  • OGC: Returns a geometric object that represents the Point set union of this geometric object with another Geometry.
See also:
http://en.wikipedia.org/wiki/Union_(set_theory)
Note:
A union of two rings can result in a polygon having a hole
Geometries:
  • polygon + box -> polygon(s)
  • ring + box -> polygon(s)
  • ring + ring -> polygon(s)
  • polygon + ring -> polygon(s)
  • combinations above -> ring(s). If the output is an ouput iterator of rings, holes are omitted
