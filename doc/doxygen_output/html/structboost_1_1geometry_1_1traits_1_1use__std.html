Detailed Description

template<typename Geometry>
struct boost::geometry::traits::use_std< Geometry >

Traits class, optional, indicating that the std-library should be used.

The default geometry (linestring, ring, multi*) follow std:: for its modifying operations (push_back, clear, size, resize, reserve, etc) If they NOT follow the std:: library they should specialize this traits class

Geometries:
  • linestring
  • linear_ring
Specializations should provide:
  • value (defaults to true)
