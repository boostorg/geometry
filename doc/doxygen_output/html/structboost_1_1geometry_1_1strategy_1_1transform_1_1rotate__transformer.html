Detailed Description

template<typename P1, typename P2, typename DegreeOrRadian>
struct boost::geometry::strategy::transform::rotate_transformer< P1, P2, DegreeOrRadian >

Strategy of rotate transformation in Cartesian system.

Rotate rotates a geometry of specified angle about a fixed point (e.g. origin).

See also:
http://en.wikipedia.org/wiki/Rotation_%28mathematics%29
Template Parameters:
P1 first point type
P2 second point type
DegreeOrRadian degree/or/radian, type of rotation angle specification
Note:
A single angle is needed to specify a rotation in 2D. Not yet in 3D, the 3D version requires special things to allow for rotation around X, Y, Z or arbitrary axis.
