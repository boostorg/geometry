Detailed Description

template<typename Geometry>
struct boost::geometry::traits::clear< Geometry >

Traits class, optional, might be implemented to clear a geometry.

If a geometry type should not use the std ".clear()" then it can specialize the "use_std" traits class to false, it should then implement (a.o.) clear

Geometries:
  • linestring
  • linear_ring
Specializations should provide:
  • apply
