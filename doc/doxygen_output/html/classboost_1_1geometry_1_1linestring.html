Detailed Description

template<typename P, template< typename, typename > class V = std::vector, template< typename > class A = std::allocator>
class boost::geometry::linestring< P, V, A >

A linestring (named so by OGC) is a collection (default a vector) of points.

Template Parameters:
P point type
V optional container type, for example std::vector, std::list, std::deque
A optional container-allocator-type (see http://accu.org/index.php/journals/427#ftn.d0e249 )
Concepts:
All algorithms work on ranges, based on a container with point types fulfilling the point concepts. They will take linestring, but also vector, std::pair, or other containers.
